PetLife v8 Eclipse 執行方式

1. Eclipse 匯入：File > Import > Existing Maven Projects
2. 選擇 PetLife 資料夾
3. 修改 src/main/resources/db.properties 的 username/password
4. 執行 controller.Main

若要產生可雙擊 JAR：
1. 在專案資料夾開 CMD
2. 執行：mvn clean package
3. 產生：target/PetLife_v8_Ultimate.jar
4. 執行：java -jar target/PetLife_v8_Ultimate.jar

注意：電腦需安裝 JDK 11 與 Maven，MySQL 8.0 需啟動。
